常見問題

  • 如何還原購買？　開啟 App →「設定 → 升級 / 支持」→ 點「還原購買」；或在付費提示頁點「還原購買」。
  • 如何取消訂閱？　訂閱由 Apple 管理：iPhone「設定 → 你的 Apple ID → 訂閱項目」中即可取消。
  • 資料存在哪裡？　你的任務與記錄目前僅儲存在裝置本地，不會上傳伺服器。
  • 退款？　付款與退款由 Apple 處理，請透過 reportaproblem.apple.com 申請。

よくある質問

  • 購入の復元方法は？　アプリを開き「設定 → アップグレード / サポート」→「購入を復元」をタップ。
  • サブスクリプションの解約方法は？　サブスクリプションはAppleが管理します。iPhoneの「設定 → Apple ID → サブスクリプション」から解約できます。
  • データはどこに保存されますか？　タスクや記録は現在、端末内にのみ保存され、サーバーには送信されません。
  • 返金について　支払い・返金はAppleが処理します。reportaproblem.apple.com よりお申し込みください。
